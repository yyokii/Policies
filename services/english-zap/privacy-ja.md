# English Zap プライバシーポリシー

本アプリ「English Zap」は、個人が提供するサービスです。お客様のプライバシーを尊重し、以下のとおり取り扱いを定めます。

1. **収集する情報**
- **検索・入力したテキスト**: 検索および AI による意味・例文の提供のため、第三者（OpenAI）の API に送信されます。OpenAI のプライバシーポリシーが適用されます。
- **保存した単語・メモ・タグ等**: お客様の端末および、お客様が iCloud を有効にしている場合、Apple の iCloud（CloudKit）上に同期・保存されます。
- **設定等**: テーマや発音設定などは、お客様の端末内に保存されます。
- **購入情報**: 有料プラン（Pro）の購入・復元は Apple が処理し、Apple のプライバシーポリシーが適用されます。

2. **アナリティクス**
現時点で、利用状況の分析のための第三者ツール（アナリティクス）は使用していません。将来、サービス改善等の目的で使用する可能性があります。その場合も、収集するデータと目的は本ポリシーに記載します。

3. **変更**
本ポリシーは、法令やサービス変更に応じて更新することがあります。重要な変更は、アプリ内や公式の告知手段でお知らせします。
